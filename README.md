# Feature Selection Pipeline

Enterprise-style ML pipeline implementing forward selection and backward elimination.

## Structure
- `main.py`: pipeline entrypoint.
- `forward_selection.py`: forward search logic.
- `backward_elimination.py`: backward elimination logic.
- `helper.py`: evaluation utilities.
- `my_plots.py`: plotting utilities.
- `configs/`, `scripts/`, `docs/`: operational scaffolding.

## Quickstart
```bash
./scripts/bootstrap.sh
./scripts/run_pipeline.sh
```
